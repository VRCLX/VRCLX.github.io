# Default VRSL Fixture Definitions

This document outlines the default VRSL fixture definitions included with VRSL. These definitions provide the channels required to control a variety of common lighting fixtures used in VRChat.
This data is a copy of data found on the official VRSL GitHub repository.

## Fixture Definitions
The following fixture definitions are included by default:
- RGB Par: 5 Channels
- Moving Head: 13 Channels
- Strobe: 5 Channels
- Flasher: 1 Channel
